What I built
  • Designed and automated a 3-layer Medallion lakehouse — ingesting raw Parquet files from Azure Blob Storage via Azure Data Factory pipelines into the Bronze layer.
  • Resolved UTF-8 encoding issues across all 9 relational source tables using Dataflows Gen2, achieving 100% schema consistency in the Silver layer.
  • Built a Gold-layer Star Schema using T-SQL Stored Procedures with incremental load logic — fact and dimension tables optimized for analytical queries.
  • Delivered sub-second Power BI reporting via Direct Lake mode, including Advanced DAX measures for time intelligence, MoM trends, and delivery KPIs.
